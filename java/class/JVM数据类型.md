# JVM数据类型

#### 基本类型：
数值类型：
`byte`（8位2的补码）
`short`（16位2的补码）
`int`（32位2的补码）
`long`（64位2的补码）
`char`（16位无符号Unicode）
`float`（32位IEEE 754单）精度FP）
`double`（64位IEEE 754双精度FP）
`boolean` 类型 （布尔值`int`将由编译器转换为布尔值，并使用相应的`int`指令）
`returnAddress`：指示指令（后者没有等效的编程语言类型）

类类型

接口类型
#### 操作指令对于类型：
`iadd`   int操作
`ladd`   long操作
`fadd`   float操作
`dadd`   double操作
